Шаг 2. Громкость микрофона

📖 Инструкция по настройке:

  1. Откройте Параметры звука Windows (правый клик на значок динамика)
  2. Выберите Свойства устройства для вашего микрофона
  3. На вкладке Уровни переместите ползунок Микрофон
  4. Добейтесь, чтобы индикатор выше показывал 🟢 зону
  5. Если громкости мало — добавьте Микрофон Boost (+10 или +20 dB)
